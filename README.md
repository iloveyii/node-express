Node Expresss
======
This is a node and express application using Typescript. It is dockerized and can be easily used for any
microservice.


## Installation
   * Using without docker
     `npm install`
     `npm start`
   * Using docker
     `docker-compose up`




## Test
   * Run test
     `npm run test`

## Tips
   * To a git repo
     `npm install --save iloveyii/ts-sequelize#node-express`
